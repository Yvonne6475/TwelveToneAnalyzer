## Twelve-Tone Music Analyzer v1.4.3

### ✨ New Features
- **Custom Merge dialog** — select arbitrary parts and bar lists (e.g. `1,3,5-10,12`) for flexible merge analysis
- **Per-bar merged set** in Merge & Search — each bar now shows the union of all parts' PCs with Forte class
- **Constituent breakdown** in Custom Merge — shows which parts contributed which pitch classes to the merged set

### 🔧 Improvements
- **music21 scratch directory** now auto-created at startup — fixes "user-specified scratch directory does not exist" error on Windows
- URL prompts updated to `.wav` examples
